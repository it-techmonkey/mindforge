# MindForge — Marketing Agency Website

A premium, conversion-focused marketing agency site built with **Next.js 14 (App Router)**, **Tailwind CSS**, **TypeScript**, and **Framer Motion**.

## Tech Stack

- Next.js 14 (App Router)
- Tailwind CSS
- TypeScript
- Framer Motion
- Lucide Icons
- Mobile-first, responsive, SEO-friendly

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm start
```

## Project Structure

```
/app          — Layout, globals, page
/components   — Navbar, UI (Button, Accordion)
/sections     — Hero, Services, Process, WhyChooseUs, CaseStudies, Testimonials, CTA, FAQ, Footer
/lib          — utils (cn)
```

## Design

- **Primary:** Maroon `#703043`
- **Backgrounds:** Off-white `#EAE1E1`, White, Charcoal `#1A1A1B`
- **Accent:** Soft Lavender `#C9B7BE`
- **Typography:** Inter (body), Poppins (headings)

Sections: Navbar, Hero, Services, Process, Why Choose Us, Case Studies, Testimonials, CTA, FAQ, Footer.
